# TCGA → Machine Learning → SHAP

Mini pipeline de ciência de dados utilizando dados públicos do TCGA (via GDC),
aplicando Machine Learning e técnicas de IA Explicável (SHAP).

## Pipeline
TCGA (API GDC) → Pré-processamento → Random Forest → SHAP

## Dados
- Projeto: TCGA-BRCA
- Fonte: Genomic Data Commons (GDC)
- Dados clínicos públicos

## Tecnologias
- Python
- Pandas
- Scikit-learn
- SHAP

## Estrutura
- notebooks/: notebooks exploratórios
- src/: código modular
- results/: gráficos e métricas
